My current simple Arch Linux laptop configuration, using Hyprland, with dotfiles manaaged using [chezmoi](https://www.chezmoi.io/).
- [waybar](https://github.com/Alexays/Waybar) configuration thanks to [mechabar](https://github.com/sejjy/mechabar)
- [rofi](https://github.com/davatorium/rofi) as the app launcher
- [dunst](https://github.com/dunst-project/dunst) as the notification daemon
- [hyprlock](https://github.com/hyprwm/hyprlock) as the lock screen
- [hyprpaper](https://github.com/hyprwm/hyprpaper) as wallpaper manager
- [hypridle](https://github.com/hyprwm/hypridle) as the idle daemon
- [flameshot](https://flameshot.org/) as the screen capture tool
- [ghostty](https://github.com/ghostty-org/ghostty) as the terminal
- [tmux](https://github.com/tmux/tmux) as a terminal multiplexer
- bash as the shell

... and [rose pine](https://github.com/rose-pine) as the main color theme.

<br>

# Configuration
This dotfiles repository uses [chezmoi](https://www.chezmoi.io/) to manage the files.
If you would like to install the same configuration files on your machine, run the following command (also shown in the chezmoi.io website):

```shell

sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply redacted24

```
